# E2E Tests

## Requirements

To run the e2e test you'll need these env variables set:
  0. Create `.env` in `e2e` directory and SET:
  1. `TEMPLATE_REGISTRY_API_URL`
  2. `AUTH_TOKEN`
  3. `IMS_CLIENT_ID`=`
  4. `IMS_CLIENT_SECRET`=`
  5. `IMS_AUTH_CODE`=`
  6. `IMS_SCOPES`=`
  7. `IMS_URL`=`
  8. `ACCESS_TOKEN`

## Run

`npm run e2e`

## Test overview

The tests cover:

1. Template Registry API CRUD Operations
